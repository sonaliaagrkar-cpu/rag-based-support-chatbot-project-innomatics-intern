Customer Support Assistant using RAG

* Developed a customer support assistant using Retrieval-Augmented Generation (RAG) for retrieval-based question answering.
* Built a system that ingests information from a PDF knowledge base, converts text into embeddings, stores them in ChromaDB, and retrieves relevant context to answer user queries.

Key Functionalities

* PDF document ingestion
* Text chunking and vector embeddings
* Retrieval-Augmented Generation (RAG) pipeline
* LangGraph workflow for query processing
* Human escalation support for unresolved queries

Technologies Used

* Python
* LangGraph
* ChromaDB
* LangChain
* Sentence Transformers

Sample Support Queries

* How can I reset my password?

* What is the refund policy?

* How are complex queries escalated?

* Demonstrates RAG workflow design, routing logic, and Human-in-the-Loop (HITL) support implementation.
